Abstract

With the use of the Oriented BRIEF (ORB), Object Binary Descriptor (OBD), and Scale- Invariant Feature Transform (SIFT), this study presents a clear approach to picture match- ing. Robust feature extraction is made possible by SIFT, computational efficiency is pro- vided by ORB, and further discriminative capability is provided by OBD. Bounding box annotation is used to establish object localization, which improves the understanding of the results. Introduction

This research introduces a framework for image matching, combining three algorithms: Object Binary Descriptor (OBD), Oriented BRIEF (ORB), and Scale-Invariant Feature Transform (SIFT). It addresses challenges like viewpoint changes and occlusions, providing reliable matching and efficient computation. The framework includes object localization with bounding boxes for visual clarity. Evaluation on benchmark datasets demonstrates its resilience, accuracy, and computational efficiency, making it applicable across various computer vision applications.

Methods used

1.SIFT(Scale-Invariant Feature Transform)

2.BRIEF (Binary Robust Independent Elementary Features) Descriptor

3.ORB (Oriented FAST and Rotated BRIEF) Descriptor

4.Object Localization
